# TODOアプリ（Python & Tkinter）

## 📝 概要
シンプルなTODOアプリです。タスクの追加、削除、完了ステータスの変更、締切日フィルタ機能を実装しています。

## 🛠️ 使用技術
- Python
- Tkinter（GUI）
- SQLite（データ管理）

## 📂 ファイル構成
- `todo_app.py` ：メインのソースコード

## 🚀 実行方法
1. Pythonをインストール（Python 3.x）
2. `todo_app.py` を実行

## 📌 特徴
✅ シンプルなGUIでタスク管理  
✅ SQLiteを使ったデータベース管理  
✅ 締切日ごとのソート機能
